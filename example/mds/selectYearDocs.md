## 使用

```html
<mj-select-year :fnSelectYear="fnSelectYear"> </mj-select-year>
```

## API

| 参数         | 说明     | 类型     | 默认值   | 必填  | 返回值         |
| ------------ | -------- | -------- | -------- | ----- | -------------- |
| fnSelectYear | callback | Function | function | false | 当前选择的年度 |

## 开发目录
```js
// example 
测试目录

// packages
组件源码目录

// lib
npm 发布目录

// dist
线上展示目录
```

#### npm构建发布流程

```js

// 构建
npm run lib
// 登录
npm login
// 发布
npm publish
```
